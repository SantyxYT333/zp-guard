Sistema de BlackList

Tenemos sistema de BlackList!
Este te avisa cuando un usuario en la blacklist se une para poder banearlo o kickearlo. Puedes reportar un usuario si hace spam, si inclumple la ToS o demas con /report user

Sistema de AntiRaid

Tenemos sistema de AntiRaid!
Este sistema viene con un sistema de WhiteList, que los usuarios que agreges a ella no seran sancionados. En cambio, si no estan en la WhiteList, si el usuario supera el limite establecido, el bot lo sancionara automaticamente
Puedes configurar el AntiRaid con /antiraid [sub-comando] [limite]

Sistema de Logs

Tenemos sistema de Logs!
Tenemos un sistema de logs para cuando un usuario crea, elimina, un Rol/Canal/Categoria para que el owner lo sepa. Este puede ser establecido con /logs set-channel y eliminado con /logs delete-dates
